Avoid gen-lang-client Bug

If your API Key project name contains gen-lang-client, it may be affected by a known bug that prevents usage.
Solution: Create a new project with a custom name (e.g., "my-notebook-pptx") in Google Cloud Console, then create an API Key under that project.

How to get an API Key:

  1. Go to Google Cloud Console and create a new project (with custom name)
  2. Go to Google AI Studio
  3. Click "Create API Key" and select your newly created project
  4. Copy the generated API Key
