配置 API 密钥

请输入您的 Gemini API 密钥以开始使用此应用程序。您的密钥将安全地存储在浏览器本地，不会发送到任何服务器。

重要：免费方案限制

Gemini 免费 API 有严格的速率限制：

  • 每分钟 2 个请求（RPM）
  • 每天 50 个请求（RPD）

为了防止达到速率限制，此工具在请求之间添加了 3.5 秒延迟。对于包含许多页面的大型 PDF，这会导致较长的处理时间。

建议： 对于更快的处理，请考虑升级到付费的 Gemini API 计划，该计划提供更高的速率限制（15 RPM）。查看定价详情
