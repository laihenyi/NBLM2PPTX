Evitar el Bug de gen-lang-client

Si el nombre de tu proyecto API Key contiene gen-lang-client, puede verse afectado por un bug conocido que impide su uso.
Solución: Crea un nuevo proyecto con nombre personalizado en Google Cloud Console (ej. "my-notebook-pptx"), luego crea la API Key en ese proyecto.

Cómo obtener una API Key:

  1. Ve a Google Cloud Console y crea un nuevo proyecto (nombre personalizado)
  2. Ve a Google AI Studio
  3. Haz clic en "Create API Key" y selecciona el proyecto que acabas de crear
  4. Copia la API Key generada
